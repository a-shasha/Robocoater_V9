# Robocoater V9

Controlled stabilization baseline for Robocoater V9 antisolvent and gas quench workflows.

## Repository layout

- `antisolvent_v9/` - Antisolvent V9 operational code moved from `Automated-Spin-Coating/In-Situ/Dual/Self_Driving_V9/`.
- `gas_quench_v9/` - Gas quench V9 operational code moved from `Automated-Spin-Coating/In-Situ/Dual/Self_Driving_V9_GQ/`.
- `docs/` - Stabilization plan, known issues, and decision log.
- `validation/` - Curated validation material only.
- `training/` - Operator training plan, checklist, and stop conditions.
- `shared_notes/` - Cross-workflow notes shared between antisolvent and gas quench stabilization.

Operational Python code is preserved unchanged in this baseline.
