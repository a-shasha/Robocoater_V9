# Kathy Training Plan

## Goals

- Run the V9 preflight checks before campaign execution.
- Confirm campaign output location and naming before starting a run.
- Recognize stop conditions and preserve run evidence after a stop.
- Record observations in the approved campaign log process.

## Initial sessions

1. Repository orientation and folder layout.
2. Preflight workflow and hardware readiness checks.
3. Campaign launch, observation, and shutdown practice.
4. Stop-condition drills and documentation review.
